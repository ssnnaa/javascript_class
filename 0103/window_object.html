01_object_window(브라우저)

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다

window객체

window객체는 브라우저 객체 중 최상위 객체로서 현재 실행되고 있는 브라우저창을 나타낸다.

window객체의 속성(프로퍼티)
document
frameElement : 내가 원하는 프레임을 선택해서 원하는 정보를 가져올 수 있음 but 요즘 잘 사용하지 안흥ㅁ
innerHEight : 윈도우창 안쪽의 높이
innerWidth : 윈도우창 안의 너비
location : 현재 내가 쓰고 있는 브라우저의 로케이션 정보를 가져오거나 설정할 ㅅ ㅜ있음
name : 브라우저창 맨 위에 뜨는 이름을 가져오고 싶을 때 사용
screenX : 브라우저 바깥쪽의 X좌표
screenY : 브라우저 바깥쪽의 Y좌표
scrollX, scrollY : 브라우저의 스크롤 값을 구할 수 있음
sessionStorge : 데이터 저장
window객체의 매서드
alert(); - 윈도우창에서 경고창(메세지)를 띄울 때 사용
blur(); - 현재창에서 포커스를 제거
close(); - 현재 창을 닫기
confirm(); - 확인, 취소 버튼이 있는 확인창을 표시함
focus(); - 현재창에 포커스를 부여
moveBy(); - 현재창을 지정한 크기만큼 이동
moveTo(); - 현재창을 지정한 좌표로 이동
open(); - 새로운 창을 연다.
prompt(); - 사용자가 입력 가능한 프롬프트창을 연다.
resizeTo(); - 동적으로 브라우저의 창의 크기를 조정
scroll(); - 문서의 특정 위치로 스크롤
sizeToContent(); - 내용에 맞게 창의 크기를 조절
stop(); - 로딩 중지

팝업창은 브라우저의 팝업차단 기능으로 인해 사용자가 보지 않을 가능성이 많아 요즘 사용하지 않고 팝업레이어, 모달윈도(css로 박스 포지션 배치)를 활용하여 띄운다.

모달윈도(modal window)
모달윈도는 사용자 인터페이스(UI)디자인 개념에서 자식 윈도에서 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다
모달 윈도우는 부모창을 배경으로 어둡게 하고 자식요소가 화면의 가운데 중앙에 배치하도록 하며 오른쪽 상단에 '닫기'버튼을 만들어서 닫을 수 있도록 한다.

실습예제. 푸터영역에 있는 관련사이트(패밀리사이트) 바로가기 구현

실습예제. location객체를 사용하여 웹페이지의 정보 표시

실습예제. screen객체를 사용하여 현재 화면 크기 해상도 표시
